Timeline


 
-- Timeline(List<List<dynamic>> channels, TimelineOptions this._options)
- Each channel contains a list of animations.
- Each channel run at the same time as the others.
- For each channel, the animations will run successively .

// The animations will run one after the other (1 channel / 2 animations)
new Timeline([[anim1, anim2]], new TimelineOptions(loop:true)).start();

// The animations will run simultaneously  (2 channels / 1 animation in each channel)
new Timeline([[anim1], [anim2]], new TimelineOptions()).start();
